# tpmpy
Python library for tensor product modeling

# First initialization of project w/ uv

1. pip install uv
2. uv init
3. uv add numpy scipy matplotlib pandas jupyter
4. now, do uv run jupyter lab (**Note:** you can also just look at the start.bat file and use that)

# Setting jupyter config params

1. uv run jupyter notebook --generate-config
2. edit the global config file
